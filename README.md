# Sathu website

Static marketing site for the Sathu app.

Files:

- `index.html`: landing page
- `styles.css`: layout and visual design
- `script.js`: language switcher and reveal animations
- `strings/en.json`: English copy
- `strings/th.json`: Thai copy

Assets are stored locally in `assets/` so the site can be deployed as a standalone repo.

For local preview, serve the folder through a simple static server so the language JSON files can be loaded by the browser.
